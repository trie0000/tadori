## A. 機能（ドメイン別）

### A-1. 認証 (auth)
- ★ SharePoint Cookie 認証（FormDigest 取得、credentials:'include'）— **検証済**
- ☆ Cookie 失効検知 → 再ログイン誘導（MSAL Public Client フォールバックは設計余地のみ）

### A-2. 取り込みパイプライン (pipeline)
- ★ Sticky モード（setInterval 30秒の取り込みループ、ON/OFF）
- ★ 未処理アイテム取得（`is_ml eq true AND (rag_status null OR claimed>10分前)`）
- ★ ランダムシャッフルで claim 衝突分散（3〜5件選択）
- ★ try-claim（ETag `If-Match` 楽観ロック、412 はスキップ）— **クライアント実装済(未結線)**
- ★ 本文クリーニング（引用・署名・HTMLタグ除去、8Kトークン切詰め）
- ★ バッチ埋め込み（16件、dimensions=256）— **検証済**
- ★ embedding 列更新 + rag_status='indexed' + embedded_at
- ★ レート制限対応（指数バックオフ・リトライ）
- ★ claim タイムアウト（10分超過で再取得可能に）
- ☆ 初期一括取り込み（過去N年分のバックフィル進捗管理）

### A-3. ベクトル格納・変換 (lib)
- ★ Float16 ⇄ Base64 エンコード/デコード — **実装済(誤差<0.0005)**

### A-4. ストレージ (storage)
- ★ IndexedDB キャッシュ（message_id 主キー、付随メタ）
- ★ 起動時キャッシュ復元
- ★ 差分同期（`Modified gt <last_sync>` のみ REST 取得してマージ）

### A-5. 検索 (search)
- ★ クエリ埋め込み（256次元）
- ★ ブルートフォース cosine + L2正規化 Top-K — **実装済(未結線)**
- ★ スニペット生成 + クエリ周辺ハイライト
- ★ 結果から元メール本文を取得して表示
- ☆ フィルタ（送信者・期間・ML 別）
- ◇ 関連メール推薦（類似度ベース）/ 複数ML横断検索

### A-6. 外部連携 (integration)
- ★ Azure OpenAI 埋め込み（中継サーバ経由）— **検証済**
- ★ OWA で開く = 「`from:<送信者> received:<受信日>` をコピー + 受信箱を新タブ」
  （直リンク不可。Spira の教訓）
- ◇ LLM 後段（要約・Q&A）、社内 Chat completion が使えれば

### A-7. 設定・永続化 (config)
- ★ 設定の localStorage 保存（中継URL/デプロイ名/APIキー/対象ML/次元数/同期間隔）
- ★ 対象 ML アドレスのホワイトリスト管理

### A-8. 状態・診断・基盤 (shell/ops)
- ★ ホスト CSS シールド付きルート注入（`#tadori-root`, all:initial, 数値z-index）
- ★ トースト通知 / モーダル基盤
- ★ ビルドID表示（キャッシュ混乱対策）
- ☆ 接続診断（relay health / SharePoint 到達 / 埋め込みテスト）
- ☆ ダークモード / 文字サイズ3段階
- ☆ エラーログ表示（コピー可能）

## B. 画面・モーダル（作成が必要なもの）

### 主画面
1. ★ **検索パネル**（メイン）
   - 検索ボックス（「メール内容を意味で検索…」）
   - 件数・所要時間表示（例「1,240件中 上位20件 / 38ms」）
   - 結果リスト（件名/送信者・日時/スニペット+ハイライト/「OWAで開く」ボタン）
   - 行クリックで詳細プレビュー（右ペイン or 展開）
2. ★ **取り込み状況バッジ/パネル**（Sticky モード）
   - 稼働中/停止トグル、claimed/embedded/indexed 件数、最終同期時刻、エラー件数

### モーダル
3. ★ **設定モーダル** — 中継URL/デプロイ名/APIキー(password)/対象MLアドレス(textarea)/
   次元数(select)/同期間隔(select)。2列グリッド、ヒント文付き
4. ☆ **接続診断モーダル** — relay health・SharePoint READ・埋め込みテストを順に実行し
   ✅/❌ を表示（PoCの test-relay.html 相当をツール内に内蔵）
5. ☆ **初回セットアップ/ガイドモーダル** — 中継起動・List列の有無チェック・対象ML設定への導線
6. ☆ **初期一括取り込みモーダル** — バックフィル対象範囲指定 + 進捗バー（過去N年分）
7. ☆ **About モーダル** — バージョン/ビルドID/リポジトリリンク（Spira aboutModal 相当）
8. ☆ **エラー/ログモーダル** — 直近のエラー一覧（本文 user-select:text + コピーボタン）

### 軽量UI要素（モーダルではない）
9. ★ **トースト** — 「検索条件をコピーしました」等
10. ★ **ランチャー/起動導線** — ブックマークレットからのパネル開閉、再起動時の復元
11. ☆ **空状態 / ローディング** — skeleton、Empty(タイトル+説明+CTA)、Error(コピー+再試行)
